NERVI 360

Mixed Reality Exploration of NERVI’s Architecture

This project was a part of The Exhibition Sydney, which was held at the Tin Sheds Gallery in the University of Sydney, and later at Australia Square. This exhibition was dedicated to the architectural work of Pier Luigi Nervi, showing off his designs in a variety of different ways.

I worked in a group of three to complete the project, and my involvement was in the creation of the Virtual and Augmented Reality applications.

For the Virtual Reality project, I used Unity and C# to allow users to experience being inside the Palazzetto dello Sport. The user is able to explore the venue and admire the beautiful architecture that Nervi designed.

Moreover, the Augmented Reality project had three buildings that the user could interact with. They had the option to either, explode (show what elements make up the building), spin, or create a cross-section of the building.

Diabetes Assist

A diabetes hub which aims to help people have more control over their disease

The concept for this app came from a group project I worked on at uni. The screens were already designed, so I thought it would be good practice to try and turn it into a full functional react app.

There are some aspects that I haven’t gotten to work yet, such as the graph section shown on the left. However, the major parts of the app function relatively well. For the meals section, I used the Spoonacular API to help with the meal images and recipes.

This project helped me learn a lot about react and nodejs, and it was also the first full stack project that I managed to deploy. The frontend and backend are being hosted on GitHub and Heroku respectively.

Smart Brain

Facial Recognition Application

I wanted to learn how to use the React.js library, and so I followed a great Udemy course taught by Andrei Neagoie. In order to grasp the different aspects of the library, I was led through the process of how to build an application that recognises the faces of people in an image.

To do this, an API called Clarifai was required to do the heavy lifting of the facial recognition. The rest of the job was mainly about making sure an image could be correctly inputted for analysis by the API.

I also learned how to implement the back-end of an application using a variety of different tools, such as PSequel, Docker and Redis. This was my first exposure to creating a full stack application, and it was amazing to see how the database side of applications is established.

Day of the Dead VR

Learn about the Day of the Dead festival through a virtual reality experience

This was a group project that I worked on for university. Our brief was to use VR to create an informative and/or social experience. We decided to focus on the Mexican festival known as the Day of the Dead, as we thought it was quite interesting and it would be fun to try and replicate certain aspects of it in a virtual space.

The user is guided through different stages, and each stage has a new set of objectives for the user to accomplish. By completing these objectives, more information about the festival can be uncovered. When the user has completed the experience, they will understand the fundamental aspects that make up the Day of the Dead festival.

Unity was used in conjunction with the Google VR package to create the virtual environment, and C# was used to code the interactions throughout the experience. We used 3Ds Max to model some of the objects in the scene, with some being imported from the Unity Assets store.

Making the user read a bunch of textual information in VR isn’t the best for immersion, so we created a journey companion that follows the user throughout the experience. The companion is a decorated skull, as shown in the image, which is one of the more prominent elements of the festival.

Football Round Up

Find info about leagues, teams and players from around the world

I love playing and watching football, so I thought it would be great to create a project that would provide all the information I needed about my favourite teams all in one place.

This application uses the SportsDB api to retrieve all the relevant data, which meant I could focus my time on the overall functionality of the project. This is another full stack project, which was the first one I had coded all on my own after the completion of the Smart Brain project.

Seeing as I had already been exposed to the development of the backend in the Smart Brain project, it was much easier to do it the second time around, since it was based on the same structure.

It is probably not the most aesthetically pleasing project in terms of the colour scheme, but I decided to just try and improve my skills with React. This project allowed me to see how far I had come in relation to using React, and it was satisfying to know that I was now capable of creating small, full stack applications on my own.
